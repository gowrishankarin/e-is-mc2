Calculating Alice's results in Bob's values

Outbound
x r = γ ( x l − vt l ) x_r=%gamma(x_l-vt_l)
t r = γ ( t l − vx l / c 2 ) t_r=%gamma(t_l-vx_l/c^2)
Inbound
x r = γ ( x l + vt l ) x_r=%gamma(x_l+vt_l)

The Equation

When something travels in no time then that will result in infinite amount of energy. We are aware energy can neither be created nor be destroyed but it can transfer from one form to another. With respect to Einstein's E = MC^2 equation, there are three possibilities

  1. There exists something with infinite mass, which results in Infinite energy
  2. Light travels @ infinite speed which results in infinite energy, which is not true
  3. Something travels in no time. ie Velocity = Distance Travelled(d) / Time Taken(t). If t = 0 then E is infinite.

What is that which travels in no time? Are we agreeing that there is inifinte amount of energy out there? What is a thought experiment? From where does thought arises? Do thought travels? If it travels then @ what speed?
